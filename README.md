# Trillo
A Practice CSS project
---
#### Install modules
```javascript
npm install
```
---
#### Build CSS and Watch for changes
```javascript
npm start:watch
```
---
#### Just build CSS
```javascript
npm run build:css
```
---
#### Just run the server
```javascript
npm run devserver
```
---
#### Build CSS and run server
```javascript
npm start
```
